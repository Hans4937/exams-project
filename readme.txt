******************************
Required:

 Have MAMP installed 
 Be able to unzip a .zip file (7-Zip or Winrar)

******************************

  Step 1:
 Download the project.zip file from Wiseflow
 
  Step 2:
 Unzip the project

  Step 3:
 Place the project folder in your htdocs

  Step 4:
 Open MAMP and make sure your ports are set to:
 Apache : 8888
 MySQL  : 3306

  Step 5:
 Start the MAMP server, go into the phpMyAdmin

 Make sure you have the following information under MySQL:
  Host     : localhost
  Port     : 3306
  User     : root
  Password : root

  Step 6:
 Make a new database with the SQL statement:
 "CREATE DATASEBASE nordicgreens" without the quotations

  Step 7:
 Select the nordicgreens database and press the import. 
 Select the nordicgreens.sql file and press "go" at the buttom of the page.

  The database is now imported with the data required for the website.

  Step 8:
 Open a new tab in your browser and write: "localhost:8888/index.php" - this is the frontpage of the website.
 
  Step 9:
 Login for the erhvers tab is:
 User     : Rema1000@test.com
 Password : Test123

  Step 10:
 Enjoy
